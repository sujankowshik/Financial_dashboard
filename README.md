# Financial Dashboard

A modern financial analytics dashboard built using React, TypeScript, Tailwind CSS, and Chart.js. The application helps users visualize income, expenses, spending patterns, savings, and financial trends through interactive charts and data-driven insights.

* Financial overview dashboard

* Interactive charts and visualizations
* Budget planning and goal tracking
* Spending trend analysis
* Category-wise expense breakdown
* Transaction management
* Financial health metrics
* CSV and Excel file import

* Real-time data insights

## Installation

### Clone Repository

```bash
git clone https://github.com/sujankowshik/Financial_dashboard.git
cd Financial_dashboard
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Usage
